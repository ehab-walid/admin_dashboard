# 🖥️ Admin Dashboard

A responsive admin dashboard UI built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies, runs directly in the browser.

---

## ✨ Features

**Sidebar Navigation**
- Dashboard branding header with icon
- Upper nav: Home, Profile, Messages, History, Tasks, Communities
- Lower nav: Settings, Support, Privacy
- All nav items use inline SVG icons (Material Design Icons)

**Header**
- Search bar with submit button
- Notification bell icon
- User profile picture and display name (top right)
- Greeting section with profile picture and username
- Action buttons: New, Upload, Share

**Main Content — three-column layout**
- **Your Projects** — card grid displaying 6 project entries, each with a title, description, and three action icons (Star, Watch, Fork)
- **Announcements** — sidebar list with titled announcement entries
- **Trending** — sidebar list showing trending users and their project names

---

## 🗂️ Project Structure

```
admin_dashboard/
├── index.html        # Full dashboard markup & layout
├── dashboard.js      # JavaScript interactions
├── style.css         # Layout, grid, and visual styling
└── images/           # SVG assets (profile picture, bell, magnify icons)
```

---

## 🛠️ Built With

- **HTML5** — semantic structure and layout
- **CSS3** — grid-based responsive layout, custom styling
- **Vanilla JavaScript** — UI interactions via `dashboard.js`
- **Google Fonts** — Roboto font family
- **Material Design Icons** — inline SVG icons throughout

---

## 🚀 Getting Started

No installation needed. Just open the file in a browser:

```bash
# Clone the repository
git clone https://github.com/ehab-walid/admin_dashboard.git
cd admin_dashboard

# Open in your browser
open index.html
```

Or download the ZIP from GitHub and open `index.html` directly.
